 1. 视觉层次与焦点引导

  - F/Z 视觉路径：用户浏览网页呈 F 型或 Z 型模式，关键信息应放在这些路径上
  - 尺寸对比：最重要的元素应最大（标题、CTA 按钮）
  - 颜色对比：主要操作用高对比色（如深色背景上的亮色按钮）
  - 留白（负空间）：通过充足的留白让关键内容"呼吸"，避免视觉拥挤

  2. 设计一致性与认知负担

  - 视觉语言统一：颜色、字体、图标风格保持一致
  - 交互模式熟悉：符合用户习惯（如导航栏在顶部，logo 可点击返回首页）
  - 信息分组：相关内容用卡片、分割线、颜色区分

  3. 情感化设计与品牌个性

  - 配色心理学：
    - 蓝色 → 信任、专业（金融、医疗）
    - 红色 → 紧迫、行动（电商、促销）
    - 绿色 → 自然、成长（环保、教育）
  - 微交互：按钮悬停效果、加载动画、过渡效果（增加愉悦感）
  - 插画/图像：传达品牌故事和情感连接

  4. 内容优先与可读性

  - 字体层级：
    - H1（36-48px）→ H2（24-32px）→ Body（16-18px）
    - 行高 1.5-1.8，段落间距 1.5-2em
  - 内容扫描性：用列表、加粗、小标题帮助用户快速浏览
  - 对比度：文字与背景对比度至少 4.5:1（WCAG AA 标准）

  5. 性能与感知速度

  - 渐进式加载：先显示框架和关键内容，图片懒加载
  - 骨架屏：比转圈 loading 更友好
  - 优化首屏时间（LCP）：< 2.5 秒

  6. 移动优先与响应式

  - 拇指区域：移动端关键操作放在屏幕中下部（易触达）
  - 断点设计：375px（移动）→ 768px（平板）→ 1280px（桌面）
  - 触摸目标：最小 44x44px（Apple HIG）或 48x48px（Material Design）

  实际案例参考

  - Stripe：极简设计 + 强对比色 + 清晰 CTA
  - Airbnb：大图 + 简洁搜索框 + 情感化插画
  - Linear：深色主题 + 流畅动画 + 专业感

  快速检查清单

  设计完成后问自己：
  - 主要操作按钮是否一眼就能看到？
  - 视觉元素是否有 3-5 级层次（不过多也不过少）？
  - 配色是否不超过 3 种主色？
  - 移动端是否可用且舒适？

  ---
  核心思想：好的设计是隐形的——用户感受到的是清晰、流畅、舒适，而不是"设计"本身。让设计服务于内容和目标，而非炫技。
